תיעוד API – תשובות לשאלונים

CRUD כתובת פרמטרים ב־URL גוף הבקשה תיאור הפעולה מה מחזירה הפונקציה שגיאות אפשריות
POST /api/answers – { client_id, questionnaire_id, answers } שומר תשובות על שאלון עבור לקוח מסוים ע"י הורה 201 Created: { message } 400 – ולידציית קלט נכשלה
400 – שגיאה בשמירה
500 – שגיאה בשרת
GET /api/answers/client/:clientId clientId – מחזיר את כל התשובות של לקוח מסוים 200 OK: מערך תשובות 500 – שגיאה כללית בקבלת התשובות
GET /api/answers/questionnaire/:questionnaireId questionnaireId – מחזיר את כל התשובות של שאלון מסוים 200 OK: מערך תשובות 500 – שגיאה כללית בקבלת התשובות
GET /api/answers/questionnaire/:questionnaireId/client/:clientId questionnaireId, clientId – מחזיר את כל התשובות של שאלון מסוים עבור לקוח מסוים 200 OK: מערך תשובות 500 – שגיאה כללית בקבלת התשובות
