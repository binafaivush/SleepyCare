תיעוד API - קבלת קוד אישור OAuth מ-Zoom

CRUD כתובת פרמטרים ב־URL גוף הבקשה תיאור הפעולה מה מחזירה הפונקציה שגיאות אפשריות
GET auth/zoom/callbac?code=... code – קוד האישור מ־Zoom (Query Param) – קבלת קוד אישור OAuth שנשלח מ־Zoom לאחר הרשאת המשתמש 200 OK: מחזיר את הקוד שנשלח מ־Zoom
console.log – מדפיס את הקוד לשרת 		400 – חסר קוד אישור
