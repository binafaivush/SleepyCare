תיעוד API – Zoom Webhook Endpoint

Method כתובת Payload (דוגמה) תיאור הפעולה תשובה
POST /api/zoom 
{
  "event": "meeting.started",
  "payload": {
    "object": {
      "id": 1234567890
    }
  }
}
 נקודת קצה לקליטת Webhook של Zoom.
מתייחס לאירועים:
  • meeting.started – התחלת פגישה
  • meeting.ended – סיום פגישה
בעתיד ניתן לעדכן במסד הנתונים סטטוס פגישה בהתאם. 		200 OK – Zoom מקבל אישור שהאירוע נקלט
