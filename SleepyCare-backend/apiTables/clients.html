תיעוד API - ניהול לקוחות (ילדים)

CRUD כתובת פרמטרים ב־URL גוף הבקשה תיאור הפעולה מה מחזירה הפונקציה שגיאות אפשריות
GET /api/clients – – שליפת כל הלקוחות (ילדים) 200 OK: מערך לקוחות 404 – לא נמצאו לקוחות
500 – שגיאה כללית
POST /api/clients – { "user_id": string,
"counselor_id": string,
"child_name": string,
"child_birthdate": date (ISO),
"notes": string (אופציונלי) } 		הוספת לקוח (ילד) חדש 201 Created: לקוח חדש נוסף 400 – שדות חובה חסרים או שגיאה בהוספה
PUT /api/clients/:id id – מזהה הלקוח { "user_id": string,
"counselor_id": string,
"child_name": string,
"child_birthdate": date (ISO),
"notes": string } 		עדכון פרטי לקוח לפי מזהה 200 OK: לקוח עודכן בהצלחה 400 – שדות חסרים
404 – לקוח לא נמצא
500 – שגיאה כללית
DELETE /api/clients/:id id – מזהה הלקוח – מחיקת לקוח לפי מזהה 200 OK: פרטי הלקוח שנמחק 400 – מזהה לא תקין או שגיאה
404 – לקוח לא נמצא
